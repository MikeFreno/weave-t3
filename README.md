## This is the repo for Weave, located at weavechat.net.

Weave is a communication platform focused on privacy, security as well as specific features aimed at programmers and investors.

Tech Stack

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
